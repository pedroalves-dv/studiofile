# Design & Aesthetic Rules

- **Typography**: `Geist Sans` is default. No `font-body` classes.
- **Borders**: ALWAYS use `border-stroke` (not `border-border`).
- **Rounding**: ALWAYS use `rounded-lg` on buttons and inputs.
- **Motion**: Import from `motion/react`, NOT `framer-motion`. Prefer CSS transitions where possible.
- **Brutalist Sensibility**: Raw, confident, Editorial well Designed Interactions. No "SaaS" polish.
